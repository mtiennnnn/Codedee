# Django Project - Codedee

## What is Codedee?

Codedee (sounds like "Code Ä‘i" in Vietnamese) is a web-based coding platform using Django framework.

![image](https://user-images.githubusercontent.com/75429369/208232697-c1233394-f18b-4d16-9a08-1775ee131ad3.png)

### Prerequisites

- Python `3.8.10`
- Django `4.1.3`

## Installation

1) Install dependencies

```
pip install requirements.txt
```

2) Run

```python
python3 manage.py runserver
```

3) Staff account credential

```
test/1
```
